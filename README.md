# SVETA ART

This repository contains the SVETA ART website and the files needed to run it locally.

View your app in AI Studio: https://ai.studio/apps/5d0edae9-42b7-4052-ad60-e2d3ec9c5b8c

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
